# **Design Pattern**

Design patterns can be categorized as 3 types.
> - Creational Pattern
> - Structural Pattern
> - Behavioral Pattern

* Creational Pattern
    * Factory method
    * Abstract factory
    * Singleton
    * Prototype
    * Builder
    * Object pool
* Structual Pattern
    * Adapter
    * Bridge
    * Composite
    * Decorator
    * Facade
    * Flyweight
    * Proxy
* Behaviral Pattern
    * Chain of responsibility
    * Command pattern
    * Interpreter
    * Iterator
    * Mediator
    * Memento
    * Observer
    * State
    * Strategy
    * Template
